如何注册京东账号？

若您还没有京东账号，请点击注册，详细操作步骤如下： 

1. 打开京东首页，在右上方，点击“免费注册”按钮；

 

2. 进入到注册页面，请填写您的邮箱、手机等信息完成注册；

 

3. 注册成功后，请完成账户安全验证，来提高您的账户安全等级。

如何注册企业会员？

注册企业会员，请点击注册，具体操作步骤如下：

1.请登陆京东网站，首页上方有“免费注册”，点击“免费注册”进入用户注册的页面；

2.用户类型请选择“企业用户”，按照页面提示将信息填写完整后提交申请即可，成功提交后，我司会安排客户经理电话联系您沟通相关合作事宜，如您的条件符合我司的企业用户条件，客户经理审核通过后您就成功的成为我司的企业用户，如果审核失败，用户类型就是“个人用户”。

部分使用使用IE 6.0 xpsp1的客户（xpsp1是IE浏览器的补丁版本号），有时无法登录京东网站。
经过分析我们发现：按如下办法，可以解决登录问题。（当然xpsp1本身也存在一些安全性漏洞/隐患，所以建议您将IE升级到6.0 xpsp2或更高版本）。

确认IE的版本

如下图（A1）所示打开IE --> 帮助 --> 关于 Internet Explorer(A)
（图：A1）点击 "关于 Internet Explorer（A）" 后， 弹出的对话框如下图（A2）所示。请注意：如果你的IE版本号中，也含有 xpsp1 字样，证明你可能无法登录京东网站。（解决办法可参考后面的其他部分）
（图： A2）

取消HTTP 1.1

如下图（B1）所示，打开IE --> 工具 --> Internet 选项
（图：B1）如上操作后， 将弹出一个对话框，如下图(B2)所示。在该对话框的 “高级” 标签中，请取消"使用HTTP 1.1"和"通过代理连接使用HTTP 1.1"（即：将两处的勾选框，保持为空）。
（图：B2）

设置完成后 可重新登录

如果您使用以上办法仍无法登陆帐户的话，根据以下各IE版本的步骤操作可以帮助您彻底解决登陆问题：

IE6.0

关掉所有的IE窗口点击“开始”---打开“控制面板”--- Internet 选项---删除cookies，删除文件---勾选“删除所有脱机内容”---点击“确认”验证是否成功：回到“ Internet选项”---设置---查看文件---如果打开的页面还有文件就将其删除。

IE7.0

关掉所有的IE窗口，点击“开始”---打开“控制面板”--- Internet 选项---高级---重置。

IE8.0

关掉所有的IE窗口点击“开始”---打开“控制面板”--- Internet 选项---高级---删除个性化设---重置。
